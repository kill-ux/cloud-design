04. Operational Commands (Makefile)
make plan Validate configuration and generate execution plan against environment variables
make apply Deploy updated infrastructure changes directly to AWS account
make ssh Lookup active EC2 instance IP address in ASG and initiate SSH connection
make cluster Output ECS cluster health status and registered container instance metrics
make services Display active state and task count for all running ECS services
make destroy Teardown provisioned cloud infrastructure resources
